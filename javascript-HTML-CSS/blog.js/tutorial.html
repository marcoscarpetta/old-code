Tutorial

Follow the four simple passes belowe.

1. Get blog.js

Download the blog.js and blog_generator.js files from the github repository.

2. Include blog.js in your page

Include blog.js and blog_generator.js files in the page where you want to load the blog: 

<script type="text/javascript" src="blog.js"></script>
<script type="text/javascript" src="blog_generator.js"></script>
You also have to create an element with id="blog.js" where you want to show the blog: 
<div id="blog.js"></div>

3. Create the posts database

Create a json file like this one and an html file for each post.

4. Edit settings

Edit blog_generator.js file following comments.

Notes

Each path (json file, html files, images in posts...) can be relative to page where the blog have to be loaded or absolute.
